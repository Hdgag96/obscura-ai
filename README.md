# ObscuraAI

**Bringing Visibility to the Invisible**

ObscuraAI is an AI-powered cyber threat intelligence and SOC analysis platform that detects, analyzes and summarizes cyber threats, providing actionable insights and defensive recommendations.

## 🚀 Features (Planned)
- Log ingestion and parsing
- AI-powered alert summarization
- Threat severity scoring
- MITRE ATT&CK mapping
- Analyst recommendations
- Interactive dashboard

## 🛠️ Tech Stack
- Python
- Streamlit
- Pandas
- OpenAI API
- SQLite
